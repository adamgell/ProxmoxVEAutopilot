Pause:
Stops the build after Windows OOBE finishes but before sysprep seals the image. RDP or Console in, install whatever you need, then click Resume & finalize on the job page. Max wait is 2 hours.
Heads-up — sysprep-compatible installs only. Apps you install during the pause MUST be machine-scoped; if anything registers per-user, sysprep's /generalize will hang on a GUI error dialog that nothing in the container can dismiss.
  • Prefer MSI installers run as admin, or winget install --scope machine <pkg>.
  • Don't launch Edge, Store, or any Windows app during the pause — first launch silently provisions a per-user copy.
  • If a per-user app slips in: Get-AppxPackage -AllUsers <name> | Remove-AppxPackage -AllUsers before clicking Resume.
