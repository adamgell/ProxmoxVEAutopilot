Setup Lifecycle

The console installer does most of the stack creation and moves the environment through three customer-visible phases.

Foundation

Prepare Proxmox access, storage, networking, and the Ubuntu controller VM.

Bootstrap

Start the controller stack, satisfy media requirements, and publish setup state.

Operational

Build and promote artifacts, verify services, and make /setup ready.
